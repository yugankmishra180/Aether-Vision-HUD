# AetherVision

A real-time cyberpunk-style AR HUD using MediaPipe and OpenCV.

## Features
- Face mesh tracking with animated landmarks
- Hand gesture detection (fist, scan, peace, target)
- Cyberpunk HUD overlays: rectangles, circles, lines, hexagons
- Dynamic particles and neural visualization
- Real-time FPS and augmentation progress bar

## Requirements
- Python 3.10
- Libraries:
  - opencv-python
  - mediapipe==0.10.9
  - numpy

## How to run
1. Activate your virtual environment
2. Run `python aethervision_hud.py`
